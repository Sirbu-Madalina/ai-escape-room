# AI Escape Room

An interactive, browser-based escape room game where every puzzle is generated or shaped by AI. Race against the clock across three themed rooms: riddles, suspicious research emails, and logic deduction.

See it live here: https://ai-escape-room-1.onrender.com/

---

## How It Works

Players start in a lobby and unlock rooms by solving puzzles. Each room has its own theme, time limit, and puzzle type.

| Room | Theme | Puzzle Type | Time Limit |
|------|-------|-------------|------------|
| 1 - Cyber Lab | Cyan | AI-generated riddle | 120 seconds |
| 2 - Research Emails | Gold | Inbox investigation | 150 seconds |
| 3 - Neon Grid | Pink | Logic deduction board | 180 seconds |

Lives are shared across all rooms. Wrong answers, answer reveals, and timeouts cost a life. Hints are free.

## Tech Stack

**Frontend**
- Vue 3 + TypeScript
- Vite
- Vanilla CSS

**Backend**
- Node.js + Express
- Socket.IO for co-op sessions
- OpenAI structured puzzle generation
- Optional MongoDB-backed session persistence

## Getting Started

### 1. Configure the backend

Create `backend/.env`:

```env
OPENAI_API_KEY=sk-...
PORT=5001
MONGODB_URI=...
MONGODB_DB_NAME=ai_escape_room
```

MongoDB is optional. Without it, sessions run in memory.

### 2. Install dependencies

```bash
cd backend && npm install
cd ../frontend && npm install
```

### 3. Run the app

Open two terminals:

```bash
cd backend
npm run dev
```

```bash
cd frontend
npm run dev
```

Then open `http://localhost:5173`.

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/riddle` | Generates an AI riddle |
| `POST` | `/api/email-investigation` | Generates a research inbox investigation |
| `POST` | `/api/logic-board` | Generates a logic deduction puzzle |

---

## Puzzle Types

### AI Riddle
A text riddle generated by OpenAI. The player submits a short answer.

### Research Emails
A suspicious company inbox with searchable emails, an employee profile, hidden clues, and a final override code assembled from multiple pieces of evidence.

### Logic Board
Three logical clues point to one correct terminal: alpha, beta, or gamma.

## Features

- Solo play and shared co-op sessions
- Shared co-op state, chat, presence, and live draft sync
- Intensity levels: Simple, Medium, Hard
- Ambient music controls with volume
- Per-room countdown timer
- Hint and answer reveal flow
- Fallback puzzles when AI generation fails
