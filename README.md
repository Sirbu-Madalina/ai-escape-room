# AI Escape Room

An interactive, browser-based escape room game where every puzzle is uniquely generated by AI. Race against the clock across three themed rooms, each with a different puzzle mechanic - riddles, Sudoku, and logic deduction. You only get three lives shared across all rooms, so every wrong answer counts.

See it live here: https://ai-escape-room-1.onrender.com/
---

## How It Works

Players start in a **lobby** and unlock rooms by solving the puzzles inside them. Each room has its own theme, time limit, and puzzle type. Lose all three lives and it's game over.

| Room | Theme | Puzzle Type | Time Limit |
|------|-------|-------------|------------|
| 1 — Cyber Lab | Cyan | AI-generated riddle | 120 seconds |
| 2 — Vault Core | Gold | 4×4 Sudoku | 150 seconds |
| 3 — Neon Grid | Pink | Logic deduction board | 180 seconds |

**Lives system:** You have 3 lives shared across all rooms. You lose a life for a wrong answer, revealing the answer, or running out of time. Hints are free.

## Tech Stack

**Frontend**
- Vue 3 (Composition API) + TypeScript
- Vite build tool
- Vanilla CSS with BEM naming and theme classes

**Backend**
- Node.js + Express 5
- OpenAI API — `gpt-4o-mini` for structured logic puzzles, `gpt-4o-mini` for riddles
- Procedural generation for Sudoku (no API call needed)

## Getting Started

### Prerequisites

- Node.js 
- An OpenAI API key

### 1. Clone the repo

```bash
git clone <your-repo-url>
cd ai-escape-room
```

### 2. Configure the backend

```bash
cd backend
cp .env.example .env   # or create .env manually
```

Add your OpenAI key to `backend/.env`:

```
OPENAI_API_KEY=sk-...
PORT=5001
```

### 3. Install dependencies

```bash
# Backend
cd backend && npm install

# Frontend
cd ../frontend && npm install
```

### 4. Run the app

Open two terminals:

```bash
# Terminal 1 — backend (http://localhost:5001)
cd backend && npm start

# Terminal 2 — frontend (http://localhost:5173)
cd frontend && npm run dev
```

Then open your browser to `http://localhost:5173`.

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/riddle` | Generates an AI riddle with difficulty & theme params |
| `GET` | `/api/sudoku` | Returns a procedurally generated 4×4 Sudoku puzzle |
| `POST` | `/api/logic-board` | Generates a logic deduction puzzle with 3 clues and 3 options |

All endpoints return a JSON puzzle object with `title`, `riddle`, `hint`, `explanation`, and the answer or solution.

---

## Puzzle Types

### AI Riddle
A text-based riddle generated by OpenAI. The player types a free-text answer, validated by a case-insensitive string match.

### Sudoku
A 4×4 grid with 6 hidden cells. The backend generates a valid solution, randomizes the digit mapping and row/column order, then hides cells. Validation checks rows and columns for uniqueness.

### Logic Board
Three logical clues point to one correct terminal (alpha, beta, or gamma). Generated by OpenAI with strict JSON schema validation and up to 3 retries. A hardcoded fallback activates if generation fails.

## Features

- AI-generated puzzles on every new game — no two playthroughs are the same
- Shared lives system creates tension across all three rooms
- Hints available for free; revealing the answer costs a life
- Per-room countdown timer; timeout triggers a new puzzle and costs a life
- Room themes (cyan / gold / pink) with matching UI styling
- Fallback puzzle system ensures the game always works even if the API is unavailable
